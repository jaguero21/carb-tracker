# CarpeCarb TestFlight External Public Beta

Use this version for external tester groups in App Store Connect.

## Beta App Description
CarpeCarb is a simple carb tracking app for iPhone. Describe what you ate in plain language and get AI-powered carb estimates, track your daily total against a goal, and optionally sync with Apple Health and iCloud.

## What to Test (App Store Connect)
Please test AI food lookup accuracy, manual entry flow, daily goal/reset behavior, saved foods, Apple Health sync, widget updates, and iCloud cloud sync across devices. Report any crashes, incorrect totals, sync delays, or confusing UI flows.

## Public Tester Instructions
1. Open the app and add several foods using AI Lookup.
2. Switch to Manual mode and add foods with explicit carb values.
3. In Settings, set a daily goal and custom reset hour.
4. Save foods and re-add them from Favorites.
5. If comfortable, enable Apple Health sync and confirm entries appear.
6. If using multiple devices with the same Apple ID, enable Cloud Sync and confirm updates propagate.

## Feedback Requested
Please include:

- Screenshot/video if possible

## Known Beta Limitations
- AI estimates may vary depending on food description quality.
- Cloud sync can take a short moment after app resume.
- Widget updates may not be instant in all iOS conditions.
